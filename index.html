Image loading demo

The below pages illustrate different image loading techniques. The problem being tackled is loading a large number of non-critical images simultaneously. In the example, we will be downloading all world flags generated from iso-country-flags-svg-collection.

Additionally, the solution must have good cross-browser support (IE11+), be cacheable and support gzip or Brotli compression.

Demos:

  • PNG - PNG images are loaded using <img>
  • SVG - SVG images are loaded using <img>
  • PNG w/ lazy - PNG images are loaded using <img> and loading="lazy"
  • SVG w/ lazy - SVG images are loaded using <img>
  • PNG in CSS - PNG in CSS
  • SVG in CSS - SVG in CSS
